# tit for tat
Start off Cooperative, if the individual cooperates with you in this round you continue.
If they individual cheats, you respond by cheating.
If they go to cooperation you cooperate again.
Limitations: only 2 actors, vulnerable to error noise (a failed read on cooperation cascades cheating)

# forgiving tit for tat 
How many times do you need to handle cascading cheating until you forgive to reestablish cooperation.
vulnerable to exploitation

# psychopathy tit for tat
Start off with tit for tat. Punitive and Retalitory.
If and only if you've gone n number of rounds you move to trusting forgiving tit for tat. 

# Pavlov Strategy
When i do something, if i get rewarded, i do it again.
If i lose, i switch strategy.
Exploits forgiving people in the long term.
