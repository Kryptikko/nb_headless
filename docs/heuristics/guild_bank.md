# Implementation

### MVP
Store stackable items.
Display item breakdown.
Display them for the user to be able to equip onto Characters.
Allow for the Player to gauge the possible upgrade. (show stats changes)
If a character already has an item in that slot the existing item is lost. (gold/item sink)

### V1 - Crafting
To create an item you need the following consumable types that are destroyed by the process.
- recepie
- materials
- gold fee
- another equipable item
