# Character
Represents the character a persona plays within the game.
Characters have attributes, abilities and equipment.
For now a Persona has a Character (1 to 1 relation).

## Leveling
Characters can level when reaching certain experience (xp) threasholds.
Leveling means increasing base attributes and unlocking training points (tp).
